# meli-test
Mercado Libre test

## Prerequisites

* React v18.0.0
* Node 
* npm 
* Express

## Installing packages and libraries

Install NodeJS libraries and dependencies

```bash
npm install .
```

Install API libraries and dependencies

```bash
cd api
npm install .
```

### Frontend

```bash
npm run dev
```
* PORT: 3000

### Backend

```bash
cd api
npm start
```
* PORT: 9000
